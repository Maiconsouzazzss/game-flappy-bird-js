\## Flappy Bird JS - Versão Adaptativa



\## 🎓 Contexto do Projeto

Este projeto foi desenvolvido como um desafio prático de consolidação de conhecimentos durante a graduação em \*\*Análise e Desenvolvimento de Sistemas (ADS)\*\* e o curso de \*\*Desenvolvimento Fullstack\*\*.



A proposta foi aplicar conceitos avançados de manipulação do DOM e lógica de programação em JavaScript para recriar a mecânica do clássico \*Flappy Bird\*, garantindo que o jogo fosse totalmente funcional e responsivo em diferentes dispositivos.

\## 🚀 Funcionalidades

\- \*\*Mecânica de Jogo:\*\* Salto, gravidade e geração dinâmica de barreiras.

\- \*\*Responsividade:\*\* Ajuste automático de escala para telas de baixa altura (modo paisagem).

\- \*\*Controles:\*\* Suporte para teclado (Enter) e eventos de toque/clique.



\## 🛠️ Tecnologias Utilizadas

\- HTML5, CSS3 e JavaScript.
